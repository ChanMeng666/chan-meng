## âš™ï¸ Configuration

User preferences and progress are automatically saved in:

| Platform | Configuration Path |
|----------|-------------------|

**Stored Data:**
- Module completion status
- User preferences (tour mode, display settings)
- Last interaction timestamp

> [!NOTE]
